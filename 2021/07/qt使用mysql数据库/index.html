在qt5.12以上中使用mysql除了按照https://www.codenong.com/cs106561773/这上面做完之后，接着在MySql的安装中(例：C:\Program Files\MySQL\MySQL Server 5.7\lib)复制一份libmysql.dll到qt的安装目录下(例：D:\Qt\Qt5.12.10\5.12.10\mingw73_64\bin)再使用程序不会报错。
