使用qt官方生成安装包工具

2021年7月8日 安装教程
使用qt官方生成安装包工具 生成安装包的核心命令 在C:\Qt\QtIFW-4.1.1\examples\tutorial下输入 C:\Qt\QtIFW-4.1.1\bin\binarycreator.exe --offline-only -t C:\Qt\QtIFW-4.1.1\bin\installerbase.exe -c config\config.xml -p packages NotePad.exe 具体过程参考博文： https://www.jianshu.com/p/039632e5f924
